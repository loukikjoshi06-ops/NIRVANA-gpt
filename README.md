#🧠 NIRVANA GPT
NIRVANA GPT
is a chracter level language model. 
built completely in PyTorch as a learning and research project.
The goal of this project is to deeply understand every component of LLMs.

#✨ Features
- Character-level tokenizer
- Token & positional embeddings
- Masked Multi-Head Self-Attention
- Feed Forward Network
- Layer Normalization
- Residual Connections
- Autoregressive text generation
- Model checkpoint saving/loading
- Built using only PyTorch

#📚 Current Version
NIRVANA GPT v1
- 6 Transformer blocks
- 6 Attention heads
- 384 Embedding dimensions
- Trained on TinyStories
- Generates coherent character-level text

#🎯 Project Goal
The objective of NIRVANA GPT is not just to build a language model,
but to understand the mathematics,
PyTorch implementation, 
and engineering behind GPT-style architectures from first principles.
